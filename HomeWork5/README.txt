#Задание

Используя https://postgrespro.ru/education/demodb развернуть big базу
Создать страницу /airports отображающую список аэропортов с сортировкой по по городу. Нужно отобразить всю информацию об аэропорте
Создать страницу с четырьмя фильтрами:
Дата вылета (выбор даты)
Аэропорт вылета (выпадающий список, один аэропорт)
Дата прилёта (выбор даты)
Аэропорт прилёта (выпадающий список, один аэропорт)
При входе на созданную страницу пользователь должен заполнить фильтры, нажать кнопку поиск и получить информацию из базы по рейсам
На странице с рейсами должно отображаться не более 25 записей если записей больше предлагать открыть следующую страницу

#Для работы с базой данных используется:

JDBC
Hibernate
