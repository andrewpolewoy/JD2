#Задание
    1.	Написать JSP страницу /signUp на которой будет форма для регистрации. Данные для регистрации:
        1.	Логин\Пароль
        2.	ФИО
        3.	Дата рождения
    2.	Зарегистрировать пользователя, значит сохранить данные о нём в приложении
    3.	Написать JSP страницу/signin на которой будет форма для входа. (логин\пароль). Если не нашли пользователя
        или не подошел пароль, выдать ошибку.
    4.	Написать JSP страницу /message где при GET запросе будет отображаться форма для отправки сообщения (логин кому\текст),
        при POST запросе отправляться сообщение.
    5.	Написать JSP страницу /chats на которой будут отображаться сообщения отправленные текущему пользователю.
        1.	Дата\время отправки
        2.	От кого
        3.	Текст
    6.	Подключите SecurityFilter с предыдущего слайда
